#  Bluetooth Controlled Land Drone Using 8051 Microcontroller

##  Project Overview

This project is a **Bluetooth Controlled Land Drone (Car)** developed as a hardware project during the **3rd Semester**. The vehicle is controlled wirelessly through a smartphone using a Bluetooth module. The system is built around the **8051 Microcontroller**, which receives commands from the Bluetooth module and controls the movement of the motors accordingly.

The project demonstrates the practical implementation of embedded systems, serial communication, and motor control using microcontrollers.
